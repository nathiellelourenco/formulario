Questionário


A.Na tag < input > com atributo type="text", para que serve o atributo "placeholder"?

O atributo placeholder serve para indicar ao leitor exemplo de como preencher o formulário.

B.Na tag < input > com atributo type="radio" ou type="radio", para que serve o atributo "checked"?

Checked é um atributo booleano, selecionado por padrão.

C.Na tag < input > para que serve o atributo "value"?

O atributo value serve para definir um valor inicial para o input, caso ele seja do tipo text.

D.Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

Atributo ‘name’.

E.Para que serve a tag < label >? Para que serve o atributo "for" nessa tag?

A principal função da etiqueta < label >é melhorar a acessibilidade, permitindo que os usuários compreendam facilmente qual campo de entrada se refere a cada etiqueta. O atributo "for" na tag < label >é usado para estabelecer uma associação entre o rótulo e o elemento de formulário correspondente.

F.Na tag < option > para que serve o atributo "selected"?

O atributo "selected" na tag < option > é usado para especificar qual opção de um elemento < select > deve ser exibida como selecionada por padrão quando a página é carregada.

G.Nas tags de formulário para que serve o atributo "required"?

Ao adicionar o atributo "required" a um elemento < input >, < select > ou < textarea >, você está informando ao navegador que o campo correspondente é obrigatório.

H.Na tag < form > para que serve o atributo "action"?

O atributo "action" na tag < form > é usado para especificar o URL ou o destino para onde os dados do formulário devem ser enviados quando o formulário é permitido.

I.Na tag < form > para que serve o atributo "method"? Quais são os valores possíveis desse atributo?

Na tag < form > o atributo "method" serve para especificar o HTTP que será utilizado para eviar os dados do formulário. Seus possíveis valores são o metodo 'GET' e 'POST'.

J.Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag < form >?

Na tag < form >,para evitar com que dados confidenciais ou que tenham preferências em serem ocultados da URL, se usa o valor "POST" do atributo "method". Dessa forma os dados do formulário não serão mostrados na URL.

K.Qual valor deve ser informado no atributo "method" na tag

em um formulário que será usado para fazer upload de arquivos?

O atributo “method” na tag < form > em um formulário que será usado para fazer upload de arquivos pode ter dois valores: “get” e “post”. O valor “get” é usado para enviar as informações obtidas pelo formulário como variáveis de URL e o valor “post” é usado para enviar essas informações como transação de postagem HTTP.

L.O que as tags < fieldset > e < legend > produzem?

As tags < fieldset > e < legend >, juntas sendo usadas no processo de execução, produzem um campo de um formulário, onde o usuario poderá colocar dados nesses campos para serem coletados no sistema.
